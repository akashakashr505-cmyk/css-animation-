# CSS Animation Project

A beautiful CSS animation project featuring smooth transitions and interactive elements.

## Features
- Animated spinner
- Image with fade-in effect
- Interactive button hover effects
- Responsive design

## Files
- `index.html` - Main HTML file
- `style.css` - Animation styles
- Includes placeholder image

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Hover over the button to see animations
